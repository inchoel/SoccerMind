# 배포 가이드 (로컬 → 공개 베타)

> 로컬호스트(`uvicorn --reload`)를 넘어 **실제 공개 베타 서비스**로 띄우기 위해 필요한 것들.
> 진실의 원천은 `기획서.md`·`아키텍처.md`. 이 문서는 운영/배포 관점만 다룬다.

## 현재 이미 갖춘 것 ✅

- **서비스 싱글턴** — `api/deps.py`의 `get_service()`가 `PredictionService`를 1회만 조립(요청마다 재조립 안 함, provider 캐싱 OK).
- **앱이 정적 SPA를 직접 서빙** — `api/app.py`가 `web/`를 `/`에 마운트(`html=True`). 프론트/백 분리 배포·CORS 불필요.
- **Graceful degradation** — 키 0개로도 기동(Elo+Wikipedia+템플릿 해설). 키는 환경변수로 주입.
- **TTL 디스크 캐시** — `data/cache.py`가 무료 API 한도(100/일, 10/분) 방어. loader 실패 시 만료 캐시로 폴백.
- **CI** — GitHub Actions(ruff + pytest, Python 3.11–3.13) 통과 상태.

## 베타 전 반드시 닫아야 할 것 (우선순위)

| # | 구멍 | 현재 상태 | 필요한 조치 |
|---|------|----------|------------|
| 1 | **비용 폭주** | `ANTHROPIC_API_KEY` 켜지면 `/api/predict` 1콜 = 유료 LLM 1콜. 공개 시 무제한 과금 | **레이트 리밋**(IP당 분/일) + 월 예산 상한. 또는 베타는 LLM 끄고 통계+템플릿만 |
| 2 | **무료 API 한도 소진** | football-data/api-football 100/일 공유. 캐시는 키별이라 다양한 대진이면 금방 소진 | #1 레이트 리밋이 같이 방어. 한도 초과 시 graceful 폴백 동작 확인 |
| 3 | **프로덕션 서버** | `uvicorn --reload`는 개발용(단일 워커, 자동 리로드) | `uvicorn ... --workers N` 또는 gunicorn+uvicorn worker, `--reload` 제거 |
| 4 | **시크릿 관리** | `.env` 파일(커밋 금지지만 서버에 평문) | 호스팅 플랫폼 환경변수 / 시크릿 매니저로 주입 |
| 5 | **HTTPS** | 없음(평문 HTTP) | 플랫폼 TLS(대부분 자동) 또는 리버스 프록시(Caddy/Nginx) |
| 6 | **캐시 영속성** | `data_cache/` 상대경로 → 컨테이너 재시작 시 증발 | 영속 볼륨 마운트, 또는 증발 감수(매번 재페치 = 한도 압박) |
| 7 | **DDG 차단 위험** | 프리뷰/부상속보가 DuckDuckGo HTML 스크래핑 | 데이터센터 IP는 차단당하기 쉬움. 베타선 옵트인(`previews=1`)이라 영향 적음. 실패 시 빈 결과로 graceful 확인 |
| 8 | **법적 고지** | 없음 | UI에 "예측은 참고용, 베팅 권유 아님" 디스클레이머 + 데이터 출처 ToS 확인, `LICENSE`(MIT) 파일 |

## 추천 호스팅 (난이도/비용 순)

1. **Railway / Render / Fly.io** ⭐ 추천 — `Dockerfile` 또는 `pyproject.toml` 자동 감지, 환경변수 UI, TLS 자동, 영속 볼륨 옵션. 무료~월 $5. **베타에 가장 적합.**
2. **단일 VPS**(Hetzner / Oracle 무료티어) + Caddy — TLS 자동, 완전 제어. 운영 부담↑.
3. **Vercel / Netlify** — ❌ 부적합. 서버리스라 디스크 캐시·장기 프로세스와 안 맞음.

## 베타까지 구체 작업 목록

1. **`Dockerfile` 추가** — `python:3.11-slim`, `pip install .[llm]`, uvicorn 다중 워커.
2. **레이트 리밋 미들웨어** — slowapi 또는 자체 토큰버킷(기존 캐시 패턴 재사용 가능). #1·#2 동시 방어.
3. **헬스체크 연결** — `/healthz`는 이미 있음 → 플랫폼 헬스체크에 연결.
4. **`LICENSE`(MIT) 파일 + UI 디스클레이머 한 줄**.
5. **환경변수 문서화** — `.env.example` 이미 있음 → README에 배포 섹션 추가.
6. **(선택) CORS 미들웨어** — 추후 별도 프론트 도메인 쓸 때만.
